# Projeto de Portfólio QA - Sauce Demo 🧪

Este repositório contém a documentação de testes manuais realizada no site [Sauce Demo](https://www.saucedemo.com/). O objetivo deste projeto foi aplicar metodologias de QA para identificar, documentar e gerir falhas críticas no sistema.

## 🚀 O que foi desenvolvido:
- **Planejamento de Testes:** Criação de Casos de Teste (CT) para fluxos críticos (Login, Carrinho e Checkout).
- **Gestão Ágil:** Utilização do **Trello** para organização de prioridades e ciclo de vida dos bugs.
- **Bug Reporting:** Documentação técnica detalhada incluindo ambiente, passos para reprodução, resultados esperados vs. atuais e evidências.

## 🛠️ Ferramentas Utilizadas:
- **Trello:** Gestão de quadro Kanban.
- **GitHub:** Documentação e versionamento das evidências.
- **Markdown:** Escrita técnica dos relatórios.
